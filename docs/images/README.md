# Documentation Images

This directory contains screenshots and diagrams for the project documentation.

## Recommended Images

| Image | Description | Usage |
|-------|-------------|-------|
| `dashboard-preview.png` | Main dashboard screenshot | README.md homepage preview |
| `dashboard-preview.gif` | Dashboard demo animation | Visual demonstration of features |
| `architecture-diagram.png` | System architecture diagram | ARCHITECTURE.md visualization |
| `filter-chain.png` | Filter chain flow diagram | ARCHITECTURE.md filter section |
| `k8s-deployment.png` | K8s deployment screenshot | kubernetes-integration.md |
| `monitoring-dashboard.png` | Monitoring metrics screenshot | monitoring-alerts.md |
| `ai-copilot-demo.png` | AI Copilot assistant demo | ai-copilot.md |
| `stress-test-report.png` | Stress test report screenshot | stress-test.md |

## Adding Images

1. Take screenshots from the running application
2. Optimize image size (recommended: PNG, max 800KB)
3. Use consistent naming: `feature-name-description.png`
4. Reference in documentation: `![Description](images/filename.png)`
